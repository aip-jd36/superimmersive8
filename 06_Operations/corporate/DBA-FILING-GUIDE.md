## Step-by-Step Filing Process

### Step 1: Access Texas SOSDirect Portal

1. Go to: https://direct.sos.state.tx.us/
2. Click **"File an Assumed Name"**
3. Select **"Assumed Name for a For-Profit or Professional Corporation"**

### Step 2: Enter Corporation Information

**Corporation Details:**
- Legal Name: `PMF Strategy Inc.`
- File Number: [Your PMF Strategy file number]
- Date of Incorporation: [2022 date]

### Step 3: Enter Assumed Name

**Assumed Name:** `SuperImmersive 8`

**Important:** Use exactly "SuperImmersive 8" (not "SuperImmersive8" or "Super Immersive 8")

### Step 4: Business Information

**County of Business:**
- If primary operations in Travis County (Austin) → Select Travis County
- If operating from Taipei but Texas-registered → Use registered agent's county

**Business Address:**
- Use your Texas registered office address

**Registered Agent:**
- Use PMF Strategy Inc.'s current registered agent information

### Step 5: Effective Date

**Start Date:** Select current date or future date (recommend: current date)

**Duration:** 10 years (standard)

### Step 6: Review and Submit

1. Review all information carefully
2. Verify spelling of "SuperImmersive 8"
3. Confirm contact email (you'll receive confirmation here)
4. Submit payment ($25 + processing fee)

### Step 7: Receive Confirmation

**What you'll receive:**
- Immediate: Email confirmation of submission
- 1-3 business days: Certificate of Assumed Name (PDF)

**Save these files:**
- Submission confirmation email
- Certificate of Assumed Name PDF
- File in: `06_Operations/corporate/filed-documents/`

## Annual Renewal

**Texas does NOT require annual DBA renewal.**

Once filed, the Assumed Name remains active until:
- You voluntarily withdraw it
- PMF Strategy Inc. is dissolved
- 10-year term expires (at which point you can renew)

**Action:** Set calendar reminder for 2036 to renew if still using DBA.

## If You Encounter Issues

### Common Problems

**Problem:** "Corporation file number not found"
- **Solution:** Verify PMF Strategy Inc. file number from Texas SOS records
- **Check:** https://mycpa.cpa.state.tx.us/coa/

**Problem:** "Assumed name already in use"
- **Solution:** Texas searches for exact matches. If "SuperImmersive 8" is taken, try:
  - "SuperImmersive 8" (different spacing)
  - "SI8"
  - "SuperImmersive Eight"
- **Check availability first:** Texas SOS name search

**Problem:** Payment declined
- **Solution:** Use different credit card, or try debit card
