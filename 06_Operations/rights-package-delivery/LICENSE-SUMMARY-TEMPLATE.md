# License Summary — Template

## Purpose

This 1-page PDF is the **most forwarded document** in the delivery package. Procurement teams, legal departments, and project managers scan this in 30 seconds to understand what they licensed and what they can do with it.

**Why it matters:**
- Familiar to buyers (every stock agency provides this)
- Easy to file (goes in "vendor documentation" folder)
- Scannable (no legal jargon, plain English)
- Comprehensive (all key terms on one page)

## Generation Instructions (Year 1 Manual)

### Step 1: Copy Template
Copy the template above into a text editor

### Step 2: Fill Variables
Replace all `[bracketed fields]` with actual values:

| Variable | Source | Example |
|----------|--------|---------|
| `[####]` in Order ID | Sequential order counter | 0001, 0002, etc. |
| `[####]` in Asset ID | From Safe Lane catalog | SI8-2026-0001 |
| `[Title]` | Asset title | "Neon Dreams" |
| `[Company Name]` | Buyer entity | "Acme Corporation" |
| `[End Client Name]` | If agency buying for client | "Coca-Cola APAC" |
| `[Tier 1/2]` | License tier | "Tier 1 Standard" |
| `[Amount]` | Order total | "8,000" |
| `[FPS]` | Video frame rate | "60" |
| `[MM:SS]` | Video duration | "2:15" |
| `[Genre]` | Content category | "Cyberpunk / Urban / Commercial" |
| Territory | From deal terms | "Worldwide" or "APAC only" |
| Term | From deal terms | "Perpetual" or "12-month" |
| Category (Tier 2) | Exclusivity category | "Quick Service Restaurant" |

### Step 3: Remove Inapplicable Sections
- If Tier 1: Delete "EXCLUSIVITY" section entirely
- If Tier 2 non-exclusive: Use "[If Tier 1...]" text
- If no territory restrictions: Remove territory restriction line
- If perpetual term: Remove term restriction line

### Step 4: Export to PDF
1. Copy filled template into Google Docs
2. Set font: Inter or Arial, 10pt
3. Set margins: 0.5 inches all sides
4. File → Download → PDF
5. Save as: `LICENSE_SUMMARY.pdf`

### Step 5: Quality Check
- [ ] All `[brackets]` removed
- [ ] Order ID and Asset ID match
- [ ] Permitted uses match deal terms
- [ ] Exclusivity section matches Tier 2 terms (or removed if Tier 1)
- [ ] Footer includes SI8 branding

**Time required:** 3-5 minutes per order

## Year 2-3 Automation

When building the platform, the License Summary should be **auto-generated from database fields:**

```javascript
// Pseudocode: Generate License Summary PDF
function generateLicenseSummary(orderId) {
  const order = database.getOrder(orderId);
  const asset = database.getAsset(order.assetId);

  const template = loadTemplate('license-summary.html');
  const filled = template
    .replace('[####]', order.id)
    .replace('[Title]', asset.title)
    .replace('[Company Name]', order.licensee)
    // ... etc for all fields

  const pdf = renderPDF(filled);
  return pdf;
}
```

**Storage location:** Same folder as Chain of Title, Invoice, etc.

## Design Guidelines

**Tone:** Professional, scannable, no jargon
**Length:** 1 page only (2 pages max if Tier 2 with long exclusivity terms)
**Format:** Plain text with box-drawing characters (renders in any PDF viewer)
**Branding:** Minimal (SI8 logo at footer, no graphics)
**Font:** Sans-serif, 10-11pt (Inter, Arial, Helvetica)

**DO:**
- Use checkmarks (✓) and X marks (✗) for scannability
- Use box-drawing characters (─ │ ┌ ┐) for structure
- Keep language plain English (no legalese unless necessary)
- Include support contact prominently

**DON'T:**
- Use emojis (too informal for enterprise)
- Use multiple fonts or colors
- Include marketing language ("the best AI video platform")
- Make it longer than 1 page (defeats "quick reference" purpose)
