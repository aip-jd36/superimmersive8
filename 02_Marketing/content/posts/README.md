# Post Drafts

One file per post. Organized by month.

## Naming convention

`YYYY-MM-DD-[platform]-[slug].md`

Examples:
- `2026-03-30-linkedin-tiktok-ai-labels.md`
- `2026-04-07-instagram-creator-record-launch.md`
- `2026-04-14-email-april-newsletter.md`

## File structure (inside each post file)

```
# Post Title

**Date:** YYYY-MM-DD
**Platform:** LinkedIn / Instagram / Facebook / Email
**Status:** Draft / Ready / Posted / Skipped
**Source article:** (if digest-driven)

---

## Draft

[Post copy]

---

## IG/FB Adaptation (if applicable)

[Shorter version for visual platforms]

## Notes

[Anything relevant — timing, context, paired posts]
```

## Workflow

1. Digest fires Monday → articles appear in `02_Marketing/intelligence/DIGEST-LOG.md`
2. Pick articles worth posting → create post file here
3. Add to `CONTENT-CALENDAR.md` queue with status `Draft`
4. Review/edit → change status to `Ready`
5. Post → change status to `Posted`, add date to Posted section of calendar
