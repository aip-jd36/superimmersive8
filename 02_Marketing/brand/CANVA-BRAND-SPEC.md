# SI8 Canva Brand Spec

Reference this when setting up your Canva Brand Kit and building carousel templates.

---

## Colors

| Name | Hex | Use |
|------|-----|-----|
| **Near-black** | `#1a1918` | Slide backgrounds (primary) |
| **Amber** | `#C8900A` | Accent, headlines, slide numbers, CTA |
| **Off-white** | `#FAFAF7` | Body text on dark backgrounds |
| **Pure white** | `#FFFFFF` | Headlines, emphasis text |
| **Mid-grey** | `#888888` | Subtext, captions, secondary info |
| **Cream surface** | `#F5F0E8` | Light slide variant (use sparingly) |
| **Deep amber** | `#5a3e00` | Text on amber/yellow backgrounds |

**In Canva Brand Kit:** Add all 6 as brand colors. Set `#1a1918` as default background, `#C8900A` as accent.

---

## Fonts

| Role | Font | Weight | Use |
|------|------|--------|-----|
| **Display / Hook** | Space Grotesk | Bold (700) | Slide 1 headline, section titles |
| **Body** | Inter | Regular (400) | Slide body text |
| **Body emphasis** | Inter | SemiBold (600) | Key phrases within body text |
| **Label / Tag** | Space Grotesk | Medium (500) | Slide number, source attribution |

**In Canva:** Both fonts are available in Canva's free library. Search "Space Grotesk" and "Inter" under text styles. Set as brand fonts.

**Fallbacks (if unavailable):** Space Grotesk → Poppins · Inter → DM Sans

## Logo / Wordmark

- **Full:** "SuperImmersive 8" in Space Grotesk Bold
- **Short:** "SI8" in Space Grotesk Bold, amber `#C8900A`
- **On dark slides:** SI8 in amber
- **On light slides:** SI8 in near-black `#1a1918`

## Instagram Carousel Template — 6 Slides

**Canvas size:** 1080 × 1080 px (square) — standard for IG feed posts

---

### Slide 1 — Hook

```
Background: #1a1918 (full bleed)

Top-left: SI8 label
  Font: Space Grotesk Medium 500
  Size: 18px
  Color: #C8900A
  Text: "SI8 ·" [topic tag e.g. "AI RIGHTS"]

Center (vertically weighted to top-half):
  Hook headline
  Font: Space Grotesk Bold 700
  Size: 52–64px (adjust to fit 1–2 lines)
  Color: #FFFFFF
  Max 8–10 words

Bottom-right: SI8 mark
  "SI8" Space Grotesk Bold, #C8900A, 20px

Bottom-left: "Swipe →" prompt
  Inter Regular, #888888, 16px
```

**Visual treatment:** Optional thin amber left-border accent (4px, full height, `#C8900A`)

---

### Slides 2–4 — Content

```
Background: #1a1918

Top-right: Slide number
  "02" / "03" / "04"
  Space Grotesk Bold, #C8900A, 24px, opacity 60%

Main content area:
  Section label (optional)
  Space Grotesk Medium, #C8900A, 16px, letter-spacing 2px, UPPERCASE

  Body text
  Inter Regular, #FAFAF7, 28–34px
  Line height: 1.5
  Max 3–4 lines per slide

  Key phrase (pull quote style)
  Inter SemiBold 600, #FFFFFF, slightly larger or amber underline accent

Bottom-right: SI8 mark (same as Slide 1)
```

---

### Slide 5 — SI8 Angle

```
Background: #1a1918 with subtle amber top-border (8px, #C8900A)

Content: Same layout as Slides 2–4
Topic label: "WHY THIS MATTERS" or "THE SI8 ANGLE"
Body: Connection to Chain of Title / SI8 value prop

Optional: Small amber pill/badge
  "SI8 CERTIFIED" or "CHAIN OF TITLE"
  Background: #C8900A, Text: #FFFFFF, Space Grotesk Bold 500, 14px
  Border-radius: rounded
```

---

### Slide 6 — CTA

```
Background: #C8900A (amber — breaks the pattern, draws attention)

Center:
  Headline: 1 action line
  Font: Space Grotesk Bold, #1a1918, 40–48px

  Subtext: URL or instruction
  Font: Inter Regular, #1a1918, 22px
  Text: "superimmersive8.com" or "Link in bio"

Bottom-right: SI8 mark in #1a1918

Optional: Arrow or simple icon in #1a1918
```

## Caption Template (below the post)

```
[Hook line — same as Slide 1, 1 sentence]

[2–3 lines expanding the point]

[1 line connecting to SI8 / Chain of Title]

→ Link in bio to learn more.

#AIVideo #AIRights #ChainOfTitle #ContentCompliance #AIMarketing #GenerativeAI #BrandSafety #DigitalRights
```

**Hashtag guidance:**
- Always include: `#AIVideo` `#ChainOfTitle` `#ContentCompliance`
- Rotate in: `#AIRights` `#BrandSafety` `#GenerativeAI` `#AIMarketing` `#AIAct` `#DigitalRights`
- Max 8–10 hashtags (IG algorithm penalises excessive hashtag use)
- Put hashtags at the end, separated by a line break

## FB Post Format

Same carousel works on Facebook. Post natively (don't cross-post via a tool — FB suppresses cross-posted content).

Caption can be slightly longer on FB (FB audience tolerates more text). Keep the same hashtags, fewer on FB (3–5 max).

---

## Tone Rules (All Platforms)

- **Direct.** Lead with the point, not the backstory.
- **No exclamation marks** unless quoting someone else.
- **No "I'm excited to share..."** — just share.
- **Numbers anchor credibility** — use them when available ("$1B", "12%", "7-day turnaround").
- **One idea per post.** If it needs two points, it's two posts.
